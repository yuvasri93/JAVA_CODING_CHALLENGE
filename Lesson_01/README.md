## Problem 1: Divide Two Variables

### Problem Statement

Declare two integer variables `x` and `y`, assign `20` to `x` and `6` to `y`. Perform integer division (`x / y`) and print the result.

### Java Solution

```java
    public class Main {
    public static void main(String[] args) {
        // Write your code here 
        int x=20;
        int y=6;
        System.out.println(x/y);
        
    }
}

```

## Problem 2: Convert Temperature

### Problem Statement

Declare a variable `temperature` and initialize it with `25.5` (Celsius). Print the temperature in Celsius and Kelvin by adding `273` to the Celsius value.

### Java Solution

```java
    public class Main {
    public static void main(String[] args) {
        double temperature = 25.5;
        System.out.println("Celsius - " + temperature);
        System.out.println("Kelvin - " + (temperature + 273));
    }
}
```
