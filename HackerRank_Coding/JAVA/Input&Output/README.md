# Java Stdin and Stdout I

## Problem

Read three integers from standard input (`stdin`) using the `Scanner` class and print each integer on a separate line.

## Approach

* Create a `Scanner` object to read input from the user.
* Read three integer values using `nextInt()`.
* Print each integer on a new line using `System.out.println()`.
* Close the `Scanner` after reading all inputs.

## Algorithm

1. Import the `java.util.Scanner` package.
2. Create a `Scanner` object.
3. Read the first integer.
4. Read the second integer.
5. Read the third integer.
6. Print each integer on a separate line.
7. Close the `Scanner`.

## Time Complexity

**O(1)**

## Space Complexity

**O(1)**

## Concepts Used

* Java
* Scanner Class
* Standard Input (`System.in`)
* Standard Output (`System.out`)
* Variables
* Input and Output Handling

## Learning Outcome

This problem demonstrates how to read multiple integer inputs using Java's `Scanner` class and display them using standard output. It reinforces the fundamentals of input handling in Java.

## Sample Input

```text
42
100
125
```

## Sample Output

```text
42
100
125
```
