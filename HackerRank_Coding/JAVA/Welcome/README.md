# Welcome to Java!

## Problem

Print the following two lines exactly as shown:

```
Hello, World.
Hello, Java.
```

## Approach

* Use the `System.out.println()` method to print each line.
* Since there is no input, the program directly displays the required output.

## Algorithm

1. Print `"Hello, World."`.
2. Print `"Hello, Java."`.

## Time Complexity

**O(1)**

## Space Complexity

**O(1)**

## Concepts Used

* Java
* `System.out.println()`
* Standard Output
* `main()` Method

## Learning Outcome

This problem introduces the basic structure of a Java program and demonstrates how to print output to the console using the `System.out.println()` method.

## Output

```
Hello, World.
Hello, Java.
```
