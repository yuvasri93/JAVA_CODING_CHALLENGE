# Java 1D Array

## Problem

Given an integer **N**, read **N** integers into a one-dimensional array and print each element of the array on a new line.

## Approach

* Read the size of the array using the `Scanner` class.
* Create an integer array of size **N**.
* Store each input value in the array.
* Traverse the array and print every element on a separate line.

## Algorithm

1. Import the `java.util.Scanner` package.
2. Read the integer **N**.
3. Create an integer array of size **N**.
4. Use a loop to read and store **N** integers.
5. Traverse the array using another loop.
6. Print each element on a new line.
7. Close the `Scanner`.

## Time Complexity

**O(N)**

## Space Complexity

**O(N)**

## Concepts Used

* Java Arrays
* One-Dimensional Arrays
* Scanner Class
* For Loop
* Array Traversal
* Standard Input & Output

## Learning Outcome

This problem introduces one-dimensional arrays in Java and demonstrates how to store, access, and traverse array elements efficiently using loops.

## Sample Input

```text
5
10
20
30
40
50
```

## Sample Output

```text
10
20
30
40
50
```
