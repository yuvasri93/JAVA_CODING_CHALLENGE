# Java If-Else

## Problem

Given an integer `N`, print the output based on the following conditions:

- If `N` is odd, print **"Weird"**.
- If `N` is even and in the inclusive range of **2 to 5**, print **"Not Weird"**.
- If `N` is even and in the inclusive range of **6 to 20**, print **"Weird"**.
- If `N` is even and greater than **20**, print **"Not Weird"**.

---

## Example

**Input**

```text
3
```

**Output**

```text
Weird
```

---

## Approach

- Read the integer input.
- Check whether the number is odd or even.
- Apply the given conditions using `if`, `else if`, and `else` statements.
- Print the appropriate result.

---

## Algorithm

1. Read the integer `N`.
2. If `N` is odd, print `"Weird"`.
3. Else if `N` is between `2` and `5`, print `"Not Weird"`.
4. Else if `N` is between `6` and `20`, print `"Weird"`.
5. Otherwise, print `"Not Weird"`.
